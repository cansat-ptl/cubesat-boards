This project template is the basis of an expansion board for YktSat nanosatellite.

This base project includes a PCB edge defined as the same size as the Main YktSat PCB with the connectors placed correctly to align the boards. All IO present on the YktSat Main board is connected to the project through the 0.1" expansion headers.
